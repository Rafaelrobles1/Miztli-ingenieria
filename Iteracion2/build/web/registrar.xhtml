Página de registro (Página en proceso)

En esta página puedes elegir en registrarte como usuario "Prestador" o un usuario "Estudiante".

Se inicia llenando los datos esenciales para la creación de una cuenta, que es el nombre completo del usuario, el teléfono para que haya contacto directo con el prestador o el estudiante según sea el caso, puede ser un número local o de celular, el correo electrónico es indispensable para poder iniciar sesión, también el usuario puede iniciar sesión usando el nombre (sin apellidos).

Una vez registrado el usuario, al iniciar sesión tiene la oportunidadn de editar sus datos de perfil y además complementar su información tal como su dirección, facultad (en el caso de estudiante), entre otros.
